#### 项目表

| 字段     | 字段名          | PostgreSQL类型 | 说明       |
|--------|--------------|--------------|----------|
| id     | id           | bigserial    | 主键       |
| 项目名称   | project_name | varchar(128) | 项目名称     |
| 权重     | weight       | integer      | 排序权重     |
| 状态     | status       | smallint     | 启用/禁用状态  |
| 是否验证权限 | require_auth | boolean      | 是否启用权限验证 |
| 创建时间   | created_at   | timestamptz  | 创建时间     |
| 更新时间   | updated_at   | timestamptz  | 更新时间     |
| 软删除    | is_deleted   | boolean      | 软删除标记    |
